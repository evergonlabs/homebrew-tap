# Evergon Labs Homebrew Tap

Homebrew formulae for [Evergon Labs](https://github.com/evergonlabs) tools.

## Install

```bash
brew tap evergonlabs/tap
brew install aiframework
```

## Update

```bash
brew update && brew upgrade aiframework
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `aiframework` | Universal automation bootstrap CLI for Claude Code |
